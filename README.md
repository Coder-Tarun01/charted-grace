# Compliance Desk India

Premium Chartered Accountant (CA) landing page built with modern web technologies. This project focuses on high-end aesthetics, responsive design, and providing clear information about CA services and compliance solutions.

## Features
- **Modern UI/UX**: Built with React, Tailwind CSS, and shadcn-ui.
- **Dynamic Animations**: Smooth transitions and entry effects using Framer Motion.
- **Service Search**: Interactive search bar for quickly finding specific compliance services.
- **Comprehensive Sections**: Includes Hero, Services, Pricing, Methodology, Testimonials, and Contact.
- **Responsive Design**: Fully optimized for mobile, tablet, and desktop views.

## Technologies Used
- **Frontend**: React 18, Vite, TypeScript
- **Styling**: Tailwind CSS, lucide-react (icons)
- **UI Components**: shadcn-ui, radix-ui
- **Animations**: Framer Motion
- **Data Fetching**: TanStack Query
- **Testing**: Vitest, Playwright

## Getting Started

### Prerequisites
- Node.js (v18+)
- npm

### Installation
1. Clone the repository:
   ```sh
   git clone <repository-url>
   ```
2. Navigate to the project directory:
   ```sh
   cd compliance-desk-india
   ```
3. Install dependencies:
   ```sh
   npm install
   ```

### Development
Start the development server:
```sh
npm run dev
```

### Building for Production
Create an optimized production build:
```sh
npm run build
```
